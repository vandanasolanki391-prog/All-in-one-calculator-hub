# All-in-One Calculator Hub

Static calculator website.

## Upload steps
1. Upload all files to GitHub repository.
2. Connect repository with Cloudflare Pages.
3. Deploy.
4. Replace `your-domain.com` in sitemap.xml after domain is final.
5. Replace contact email in contact.html.

## Included
Category-wise homepage, SEO meta tags, robots.txt, sitemap.xml, legal pages and calculator pages.
